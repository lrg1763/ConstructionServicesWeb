# ConstructionServices

Одностраничный сайт-визитка для строителя-отделочника: отделочный ремонт, сантехника, электрика.

## Технологии

- **HTML5** — семантическая разметка
- **Tailwind CSS** — стили (подключение через CDN)
- **JavaScript** — мобильное меню, плавная прокрутка, галерея с прокруткой и управлением с клавиатуры
- **Font Awesome** — иконки
- **Google Fonts (Manrope)** — основной шрифт

## Структура проекта

```
ConstructionServicesWeb/
  index.html      — главная страница
  css/
    styles.css    — кастомные стили (шрифт, скроллбар, фокус, галерея)
  js/
    script.js     — меню, плавная прокрутка, галерея
  README.md
```

## Секции сайта

- **Навигация** — фиксированная шапка, логотип, ссылки (Главная, Преимущества, Услуги, Галерея, О нас, Отзывы, Контакты), кнопки «Позвонить» и «WhatsApp»; на мобильных — гамбургер-меню
- **Hero** — заголовок, подзаголовок, кнопка «Получить расчёт»
- **Преимущества** — выезд на замер бесплатно, гарантия, фиксированная смета, без предоплаты до этапа
- **Услуги** — три карточки: отделочный ремонт, сантехника, электрика
- **Галерея выполненных работ** — горизонтальная прокрутка 10 фото, стрелки влево/вправо, управление с клавиатуры (стрелки)
- **Обо мне** — краткий текст о мастере
- **Отзывы** — два отзыва клиентов
- **Контакты** — телефон, ссылки на Telegram, WhatsApp, Max, Profi.ru, Mail.ru

## Как открыть

1. Откройте `index.html` в браузере (двойной клик или «Открыть с помощью»).
2. Либо запустите локальный сервер из корня проекта, например:
   ```bash
   npx serve .
   ```
   или
   ```bash
   python3 -m http.server 8000
   ```
   и перейдите по указанному адресу (например, http://localhost:8000).

## Перед публикацией

- Замените заглушки контактов в `index.html`: телефон, ссылки Telegram (`t.me/...`), WhatsApp (`wa.me/...`), Max, Profi.ru, Mail.ru.
- Добавьте файл `favicon.ico` в корень проекта (ссылка на него уже есть в `<head>`).
- В блоке JSON-LD (Schema.org) укажите реальный `"url"` страницы, если сайт будет доступен по постоянному адресу.
