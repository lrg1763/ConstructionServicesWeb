# Галерея (WebP, 3:4)

Положите сюда **10 файлов** в формате **WebP** с соотношением сторон **3:4** (портрет, ширина : высота):

| Файл       | Описание        |
| ---------- | --------------- |
| `01.webp`  | фото 1          |
| `02.webp`  | фото 2          |
| …          | …               |
| `10.webp`  | фото 10         |

Имена должны совпадать с путями в `index.html`: `images/gallery/01.webp` … `images/gallery/10.webp`.

Рекомендуемый размер по длинной стороне: **800–1200 px** (для чёткости на экранах с высокой плотностью пикселей).
